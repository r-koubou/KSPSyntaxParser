# Tab separated colmn format

## Variables

(name)

## Callback

1. has argument(s)

    (name)\t(type)(\t(type)*

2. don't have argument

    (name)

## Command

1. has argument(s)

    (return-type)\t(name)\t(type)(\t(type))*

2. don't have argument

    (return-type)\t(name)

## Type

- I   : int
- I[] : int array
- R   : real
- R[] : real array
- S   : string
- R[] : string array
- V   : void ( command not return value )
- \*   : any
